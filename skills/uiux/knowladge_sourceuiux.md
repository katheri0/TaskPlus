# Preferred Knowledge Sources

Priority order:

1. Nielsen Norman Group (NN/g)
2. Apple Human Interface Guidelines
3. Material Design 3
4. W3C WCAG 2.2
5. Figma Learn
6. Interaction Design Foundation (IxDF)
7. ISO 9241-210 Human-Centered Design
8. UXPA Competency Framework

When multiple sources disagree:
- Follow platform guidelines (Apple HIG or Material Design) for platform-specific behavior.
- Otherwise prioritize NN/g usability guidance.
- Always satisfy WCAG 2.2 accessibility requirements.
